## 📝 오늘 배운 내용 요약

### 1. IT 인프라 환경 이해
* **온프레미스 (On-Premises):** 네트워크 스토리지 등 IT 인프라를 자체 데이터 센터나 사내 전산실에 직접 설치하고 운영하는 방식

### 2. 네트워크 통신 기초
* **프로토콜 (Protocol):** 컴퓨터 네트워크 환경에서 기기들이 데이터를 원활하게 교환하기 위해 정의한 통신 규약 및 규칙의 집합
* **OSI 7 계층:** 네트워크 통신의 과정을 7단계로 표준화한 모델
* **TCP / UDP:** 데이터 전송 방식의 대표적인 두 가지 프로토콜 (신뢰성의 TCP vs 속도의 UDP)

### 3. 주요 네트워크 장비
* **L2 스위치:** MAC 주소를 기반으로 동일한 네트워크(LAN) 내의 기기 간 데이터를 연결하고 스위칭하는 장비
* **라우터:** IP 주소를 기반으로 데이터 패킷의 최적 경로를 지정해 서로 다른 네트워크를 연결해 주는 장비

#### 💻 라우터 기본 명령어 정리
```bash
enable # 관리자 모드로 전환
configure terminal # 전역 설정 모드
line console 0 # 콘솔 포트 설정 모드 진입
login # 로그인 활성화
password 1234 # 로그인 비밀번호 설정
exit # 나가기 (2번, 혹은 end)

# 관리자 모드 진입 비밀번호 설정
enable
configure terminal
enable password 1234 # 관리자 모드 진입 시 비밀번호 요구 설정
end
exit

# 설정 확인 및 패스워드 암호화
show running-config # 현재 설정 내용 및 이전에 입력했던 목록 확인
configure terminal
service password-encryption # 설정한 평문 비밀번호가 보이지 않도록 암호화 설정

4. 네트워크 상태 확인
ipconfig: 현재 PC의 네트워크 설정 확인 (서브넷, IPv4, IPv6 등)

ipconfig /all: 물리적 주소(MAC)를 포함한 PC의 전체 네트워크 세부 정보 확인
