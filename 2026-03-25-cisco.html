날짜: 2026년 3월 25일
주제: CISCO, 네트워크 논리적 분할(VLAN)

📝 오늘 배운 내용 요약

1. 스위치 VLAN 생성 및 포트 할당

물리적인 스위치를 논리적으로 나누어 그룹화하는 VLAN(Virtual LAN) 설정 방법을 학습했습니다.

  • VLAN 생성 및 이름 지정: 스위치 내에 논리적 그룹인 VLAN을 생성하고 직관적인 이름을 부여합니다.
  • 액세스 모드(Access Mode): PC가 연결된 단일 포트에 특정 VLAN을 할당하여 해당 그룹의 통신만 허용합니다.

💻 VLAN 및 Access 포트 관련 명령어

```bash # VLAN 생성 및 이름 설정 vlan 10 name Teacher vlan 20 name Student

VLAN 생성 확인

show vlan

특정 포트에 VLAN 할당 (액세스 모드)

switchport mode access switchport access vlan 10

트렁크 모드 설정

switchport mode trunk

  1. 스위치 트렁크(Trunk) 모드 개념 여러 개의 VLAN 데이터가 스위치 간, 혹은 스위치와 라우터 간에 원활하게 지나다닐 수 있도록 해주는 통로 역할을 학습했습니다.

트렁크 모드 설정: 하나의 포트를 통해 다수의 VLAN 트래픽이 이동할 수 있도록 구성합니다.

트러블슈팅 포인트: 통신이 안 될 때는 라우터 설정 및 트렁크 설정이 제대로 되었는지 show running-config 명령어를 통해 꼼꼼히 확인해야 합니다.
